# InterConnect Application
InterConnect travelAssist is a peer-to-peer platform that connects 
travelers moving between the same countries or following similar itineraries.
It allows travelers to request assistance and enables others to offer help safely and transparently.

### Required Features
Post travel assistance requests
Browse helpers by itinerary, profile, and rates
Mutual approval before sharing contact details
Ratings and reviews for trust and safety
Secure, privacy-first connections

### How It Works: 
Traveler posts a request
Helpers with matching itineraries view it
Both users agree to connect
Contact details are shared
Assistance and communication take place

### Tech Stack
Backend: .NET (C#)
Frontend: Web Application. 
Database: SQL or PostgreSQL
Cloud: AWS or  Azure

## Goal
To build a trusted global community where travelers help travelers.
